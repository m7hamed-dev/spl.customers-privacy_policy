# SPL Customers — Privacy Policy

A store-compliant Privacy Policy page for **SPL Customers**, built with
[Next.js](https://nextjs.org) (App Router), TypeScript and Tailwind CSS, and
themed from your app's branding. Generated by Flutter Pro Kit.

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build a static site

```bash
npm run build
```

The site is configured for **static export** — the command writes a
self-contained static site to `out/`, ready to host at the privacy-policy URL
Google Play and the App Store both require.

## Before publishing

Replace every highlighted `[placeholder]` in `lib/content.ts` (contact email,
company name/address, website, retention window) with your real details, and
confirm the policy matches the features your app actually uses.
